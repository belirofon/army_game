# Армейка — текстовая RPG-игра о службе в армии

Текстовая RPG-игра о службе в армии, где игрок проживает 24 месяца военной службы, принимая решения и справляясь с различными событиями.

## Описание

Игрок проходит через 24 месяца (хода) службы, где каждое событие — это испытание, проверяющее его характеристики. Успех или неудача зависят от выбора игрока и уровня развития его навыков.

### Персонажи

Игрок выбирает одного из 4 типов персонажей с разными стартовыми характеристиками:

| Персонаж | STR | END | AGI | MOR | DISC | Особенность |
|----------|-----|-----|-----|-----|------|-------------|
| **Дрыщ** | 3 | 4 | 6 | 5 | 4 | Быстрый, но слабый |
| **Качок** | 7 | 6 | 3 | 4 | 3 | Сильный, но неповоротливый |
| **Ботаник** | 3 | 3 | 4 | 5 | 7 | Умный, но физически слабый |
| **Блатной** | 4 | 4 | 4 | 7 | 6 | Харизматичный, но средний во всём |

### Характеристики

- **STR** (Сила) — физическая сила
- **END** (Выносливость) — способность выдерживать нагрузки
- **AGI** (Ловкость) — скорость и координация
- **MOR** (Мораль) — психологическая устойчивость
- **DISC** (Дисциплина) — способность следовать правилам

### Типы проверок

Игра использует 3 типа проверок:

1. **THRESHOLD** — проверка на преодоление порога (нужно набрать >= N очков)
2. **PROBABILITY** — вероятностная проверка (шанс успеха в процентах)
3. **CATASTROPHIC** — катастрофическая проверка (высокий шанс провала с серьёзными последствиями)

### Сложность

Сложность событий увеличивается по ходу игры:
- Месяцы 1-8: коэффициент 0.6 (лёгкие события)
- Месяцы 9-16: коэффициент 0.85 (средние события)
- Месяцы 17-24: коэффициент 1.05 (сложные события)

## Стек технологий

### Backend

- **Go** — основной язык
- **GraphQL** — API (gqlgen)
- **PostgreSQL** — база данных
- **Docker** — контейнеризация

### Frontend

- **Vue 3** — фреймворк
- **TypeScript** — типизация
- **Tailwind CSS** — стилизация
- **Vite** — сборка

## Запуск

### Требования

- Docker и Docker Compose
- Node.js 18+ (для разработки frontend)
- Go 1.21+ (для разработки backend)

### Быстрый старт

```bash
# Клонирование репозитория
git clone https://github.com/belirofon/army_game.git
cd army_game

# Запуск через Docker
docker-compose up --build
```

После запуска:
- Frontend: http://localhost:5173
- Backend: http://localhost:8080/graphql

### Ручной запуск

```bash
# Backend
cd backend
go run cmd/server/main.go

# Frontend (в отдельном терминале)
cd frontend
npm install
npm run dev
```

## События

Игра содержит 26 событий, основанных на реальных историях из армейской жизни:

- Столовая (dining_room)
- Казарма (barracks)
- Учебка, полигон, баня, гауптвахта и другие локации

Каждое событие имеет несколько вариантов выбора с разными последствиями для характеристик.
